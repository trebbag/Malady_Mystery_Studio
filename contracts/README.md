# Contracts

This directory contains machine-readable schemas for the platform’s major artifacts.

## Rules

- Prefer additive evolution.
- Update examples whenever you change a schema.
- Review `api/openapi.yaml` when request or response shapes change.
- Breaking changes should trigger ADR review.
