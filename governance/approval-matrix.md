# Approval matrix

| Artifact / Stage | Clinical reviewer | Editorial reviewer | Product/production reviewer | Security/privacy reviewer |
|---|---|---|---|---|
| Disease packet | Required | Optional | Optional | As needed |
| Story workbook | Required | Required | Optional | As needed |
| Scene cards | Recommended | Required | Optional | No |
| Panel plans | Recommended | Required | Recommended | No |
| Release bundle | Required | Required | Required | As needed |

## Notes

- Publishable content should not bypass clinical review.
- A release bundle should not be exported without the minimum required approvals.
- If future customer environments introduce sensitive data, security/privacy review becomes mandatory for relevant runs.
