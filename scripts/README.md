# Scripts

Utility scripts for validating and maintaining the starter pack.

## Current scripts

- `validate_repo_pack.mjs` — validates required files, JSON schemas, examples, datasets, and workflow/OpenAPI configuration.
- `validate_repo_pack.py` — compatibility wrapper that forwards to the Node validator.
- `lint_repo.mjs` — runs the repository lint baseline by syntax-checking JavaScript modules.
- `build_workspaces.mjs` — creates a lightweight `dist/` bundle for the executable starter services and contracts.
