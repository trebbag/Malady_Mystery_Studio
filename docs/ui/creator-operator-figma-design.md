# Creator-Operator Figma Design

The current creator-operator UI direction lives in Figma:

[Malady Mystery Studio - Creator Operator UI](https://www.figma.com/design/6fbXBRFKg3ZUVzY5BkCdh4)

## Screen Set

- `01 Home / Start and resume`
- `02 Run Overview / Guided workspace`
- `03 Guide Review / Pre-render gate`
- `04 Render Panels / Create final art`
- `05 Export / Package local files`

## Implemented Direction

- Warm sand workspace instead of a dark dashboard canvas.
- Narrow dark creator rail with only the primary destinations.
- Cream tactile cards with soft borders and restrained shadows.
- Cyan primary actions for the one next step.
- Plain-language labels such as `safety checks` and `completed panel set` before technical terms.
- Advanced technical views remain available but are visually secondary.
